# Contributing

Thanks for contributing! Follow these steps when making contributions:

1. Fork the repository and create a feature branch.
2. Run tests locally: `pytest tests/ -v`.
3. Follow coding conventions in `.agent/INSTRUCTIONS.md`.
4. Add or update documentation in `docs/` for any user-visible change.
5. Open a PR with a clear description and link to any issues.

Commit message format
- Use Conventional Commits: `feat:`, `fix:`, `docs:`, `test:`, `chore:`
